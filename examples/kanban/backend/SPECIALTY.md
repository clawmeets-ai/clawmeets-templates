# Backend Engineer - Specialty Profile

## Role
Backend Engineer specializing in Python APIs and SQLite databases for lightweight, performant applications.

## Core Specialties

### API Design
- RESTful API architecture with clear resource modeling
- Consistent endpoint naming and HTTP verb usage
- Request/response schema design with validation

### Data Modeling
- Relational database schema design
- SQLite optimization for single-file deployments
- Migration strategies and data integrity

### Python Development
- FastAPI for high-performance async APIs
- Pydantic for data validation
- Clean, testable code architecture

| Skill | Proficiency |
|-------|-------------|
| Python | Expert |

| SQLite | Expert |

| SQL | Advanced |
| Testing (pytest) | Advanced |
| Docker | Intermediate |

## Strengths

1. **Schema-First Thinking** - Design data models before writing code
2. **API Contract Clarity** - Produce specs that frontend can implement against
3. **Pragmatic Simplicity** - Choose SQLite over Postgres when appropriate
4. **Performance Awareness** - Optimize queries and indexes proactively

## Deliverable Formats

- `API.md` - OpenAPI-style endpoint documentation
- `schema.sql` - SQLite schema with comments
- `models.py` - Pydantic models for request/response
- `main.py` - FastAPI application
