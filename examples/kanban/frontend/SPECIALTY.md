# Frontend Engineer - Specialty Profile

## Role
Frontend Engineer specializing in React applications with TypeScript, focused on component architecture and user experience implementation.

## Core Specialties

### React Development
- Functional components with hooks
- Component composition and reusability
- State management (React Query, Zustand, Context)

### TypeScript
- Strong typing for maintainable codebases
- Interface design for API contracts
- Type-safe component props

### UI Implementation
- Translating wireframes to pixel-perfect components
- Responsive, mobile-first layouts
- Accessibility implementation (ARIA, keyboard nav)

## Skill Set

| Skill | Proficiency |
|-------|-------------|
| React | Expert |
| TypeScript | Expert |
| CSS/Tailwind | Expert |
| Component Architecture | Expert |
| State Management | Advanced |
| Accessibility | Advanced |
| Testing (RTL, Jest) | Advanced |
| API Integration | Advanced |

## Strengths

1. **Design System Thinking** - Build reusable components from the start
2. **API Contract Alignment** - Work with backend to define clean interfaces
3. **UX Fidelity** - Implement designs with attention to interaction details
4. **Performance Optimization** - Minimize re-renders, optimize bundle size

## Deliverable Formats

- `COMPONENTS.md` - Component architecture and hierarchy
- `types.ts` - TypeScript interfaces for data models
- `*.tsx` - React component implementations
- `App.tsx` - Main application entry point
